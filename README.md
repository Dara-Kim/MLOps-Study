# MLOps-Study  

AI와 빅데이터의 시대인만큼 현업에서 AI를 효과적으로 운영하고 적용하는 데 필수적인 MLops의 중요성이 커지고 있습니다.

### 🚀MLops란?

ML 시스템 개발(Dev)과 ML 시스템 운영(Ops)을 통합하여 데이터 레이블링의 일관성을 유지, 자동 운영하는것<br/>  
즉, 계속해서 AI/ML을 개발, 배포, 운영하는 Workflow 자체를 MLops라 함
